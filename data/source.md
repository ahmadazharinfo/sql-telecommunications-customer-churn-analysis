# Data Source

Original flat file: IBM Telco Customer Churn dataset.
Source: https://www.kaggle.com/datasets/blastchar/telco-customer-churn
7,043 customers, 21 columns, no ML processing applied.

The 4 CSVs in this folder (customers.csv, phone_services.csv, internet_services.csv, accounts.csv) are the result of normalizing that single flat file into a relational schema. See `/docs/schema_notes.md` for the transformation logic.
